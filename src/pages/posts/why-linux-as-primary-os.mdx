## What is a Homelab?

Homelabbing is how you learn about servers without any risk of breaking things in a production environment at an IT job you have.

## Why should you install Linux?

I use Linux because it is a very common operating system for servers.
There are many distributions, or flavors, of Linux, and the one you choose doesn't make too big of a difference in what you are able to do with your homelab.
Linux distros are all UNIX at their core, which is a command-line only operating system. There are still command line versions of Linux, which are typically branded as server operating systems, but there are many very popular flavors with graphical interfaces.
I chose to install Ubuntu Linux as my primary OS in the spring of 2021, after using Windows for many years. I also have a macbook that I use when traveling, but I am just talking about my home desktop PC here.
I was in college and knew I would be soon working in the software and IT industry, and I had always heard that linux was a valuable skill to know about. I had been exposed to Linux over the years, from installing Kali linux to learn about ethical hacking, to using raspberry pi's to make some lights blink. I had been considering switching to Linux full-time for a while, and my desire to dive deeper into homelabbing finally prompted the leap.
When selecting my flavor of Linux, my research basically narrowed it down to two options: Debian and Ubuntu (or uboontu). These are the most popular and stable OS's. I ultimately went with Ubuntu and haven't looked back.
Coming from Windows, I noticed that Linux was going to be a change. Since it is an entirely different operating system, the user interface took some getting used to, and some software was going to take a bit more effort to get working. It just didn't feel as comfortable as what I was used to.
In summary, I think that if you are considering learning about Linux, the best way to do that is to be hands on, immersed in it for your daily usage. I bought a 256GB ssd and installed Ubuntu on it, which still gave me plenty of space to install software and store some files. I am still using the same ssd today, and and it's working great. Granted I have much more data than that, but it is stored on other drives in my computer or on my network. I recommend that you also take this approach, because then you still have your Windows OS installed on another drive in your computer, and can switch back at any time. Just reboot and select the drive in your boot manager.
Disclaimers:
It is not all sunshine and rainbows. You have to be willing to encounter a bit more resistance in most of the things you're used to doing. To install software, there are no .exe files, so you'll need to memorize or write down a couple simple commands. To play games, you need to enable a special setting in steam to allow games to work on Linux.
Running into speed bumps like these should be expected when learning anything new, and I believe Linux is something worth learning and you shouldn't let these things discourage you.
